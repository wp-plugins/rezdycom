== Description ==
Reservation plugin using the rezdy.com service

== Installation ==
1. Upload `rezdy` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your Rezdy alias or url under the 'Rezdy' settings page
